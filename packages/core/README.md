# @skastr0/background-tasks-core

Core task lifecycle, log, CLI-store, and control-plane primitives for `background-tasks`.

This package is part of the `background-tasks` workspace and is intended for Bun-based agent workflow tooling. The project is experimental; public APIs may change during `0.x` releases.

## Install

```sh
npm install @skastr0/background-tasks-core
```

## Exports

- `@skastr0/background-tasks-core`
- `@skastr0/background-tasks-core/cli-store`
- `@skastr0/background-tasks-core/control-plane`
- `@skastr0/background-tasks-core/agent-cli`

See the repository README for CLI usage, security boundaries, and release status.
